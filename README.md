# Kenetik Growth System — Dashboards

Live operational dashboards for the Kenetik Growth System. These are auto-published via GitHub Pages whenever a process is built or a status is updated.

## Dashboards

- **[Automation Architecture Dashboard](kenetik-growth-system-dashboard.html)** — Full system state: process statuses, infrastructure connections, build patterns
- **[Engineering Roadmap](kenetik-engineering-roadmap.html)** — Implementation waves, engineering dependencies, bridge strategy, cost phasing

## How These Stay Updated

Dashboard updates are pushed automatically at the end of every `/build` and `/update-status` session in the Kenetik Growth System plugin. The source of truth lives in the private `kenetik-growth-system` repo; this public repo mirrors only the rendered dashboards.
